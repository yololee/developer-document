# Mac快捷键

## mac

* option+command+c 获取绝对路径
* command + tab 切换应用
* command + ` 同一个app切换应用
* control + command + q 锁定屏幕 
* command + m 应用程序最小化
* Command + n 打开访达
* command + option + v 剪切 (需要先command + c)
* command + shift + 句号（显示隐藏文件在按一遍继续隐藏）

## IDEA

### 查找替换

* command+F  当前文件查找

* command+shift+F 全局查找

* command +shift+R  全局替换

* double shift 查询任何东西

### 注释

* command+/  行注释

* command + option + / 块注释

### 编辑

* option + O 优化import
* option+command+L  格式化代码
* command + D 复制当前行或选定的块
* command + delete 删除当前行或选定的块
* command + option + T 包围代码(if,try catch)
* command + N 生成get、set方法
* control + I 实现接口方法
* control + O 重写父类方法

### 热部署

command + shift + fn + f9

## typora

- **插入表格**：`option` + `command` + `T`
- **无序号**：`-` 或 `*`
- **有序号**： 阿拉伯数字后加上 `.` 后直接 ”空格“
- **文本加粗**：`command` + `B`
- **文本斜体**：`command` + `I`
- **文本下划线**：`command` + `U`
- **标题**：`command` + `1` ；`command` + `2` ……；`command` + `7`

## Iterm

* 新建标签  `command` + `t`
* 查找 `command` + `f`
* 垂直分屏：`command` + `d`
* 水平分屏：`command` + `shift` + `d`
* 到行首：`ctrl `+ `a`
* 到行尾：`ctrl` + `e`
* 清除当前行：`ctrl` + `u`
